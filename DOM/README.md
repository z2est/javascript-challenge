# DOM Challenge

## Q1) prompt로 입력받은 텍스트로 List를 생성해보세요.

1. `prompt`를 사용하여 사용자로부터 리스트 항목의 내용을 입력받습니다.
2. 입력받은 텍스트를 `<li>`로 감싸서 `<ul>`에 추가합니다.
3. 사용자가 `Esc` 키를 누르거나 취소 버튼을 누르기 전까지는 계속 입력할 수 있습니다.
4. 확인 버튼을 누르면 화면에 렌더링합니다.
